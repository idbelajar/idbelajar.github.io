14. An example of 'if clause' an impimperative is


If Clause as an ImperativeJawabanPendahuluan

If clause + an imperative adalah suatu kalimat untuk memberikan perintah atau saran. Artinya jika kondisi pertama terjadi, maka kondisi kedua harus dilakukan. Bentuk dari if clause as an imperative :

                       IF +SIMPLE PRESENT, VERB 1+... (imperative)

Pembahasan

Contoh kalimat if clause as an imperative :

-  If you go to the market, buy some tomatoes.

- If you see Anita, tell her I am waiting here.

- If you go to South Korea, bring me some souvenir.

- If you tired, go to sleep.

- If you arrive at home, call me.

- If you come early, open the door for me.

- If you sick, get a sick certificate.

- If you get home, send me a letter
