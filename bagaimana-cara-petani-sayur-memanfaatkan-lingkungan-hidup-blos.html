BIOS atau BLOS kak?

kalau BLOS saya belum pernah dengar

1. BIOS (Basic Input/Output System) adalah program yang digunakan mikroprosesor komputer untuk memulai sistem komputer setelah dinyalakannya. Ini juga mengelola aliran data antara sistem OS (Operating System) dan perangkat yang terpasang seperti hard disk, adaptor video, keyboard, mouse, dan printer.

2. BIOS pada hard disk digunakan untuk mendeteksi hardisk yang kita pasang. Teknik dasar BIOS untuk hardisk sendiri adalah mendeteksi adanya hardisk atau tidak, selain itu BIOS juga bisa kita atur untuk melakukan booting dari hardisk.

3. biasanya BIOS berada di bagian bawah HDD (Hard Disk Drive)

1. Cara Setting Bios Secara Lengkap

Sebelum menginstalasi sistem operasi pada komputer, pertama-tama kita harus menginstal bios yang ada di komputer agar sistem komputer mengenali CD instalasi pada CD ROOM. Akan tetapi, untuk mencegah terjadinya file yang coroupt, maka proses penyalinan data selama instalasi lebih baik jika CD menggunakan mode ROOM.

Dalam konteks ini, kami akan memberikan cara setting bios atau untuk mengkonfigurasi BIOS Anda sebelum menginstal sistem operasi di komputer Anda. Tetapi sebelum membaca panduan ini, alangkah baiknya jika Anda pertama kali memahami apa singkatan dari BIOS dan apa definisi bios.

BIOS adalah singkatan dari Basic Input Ouput System, sebuah Bios adalah program mikroprosesor untuk komputer pribadi yang digunakan untuk mem-boot sistem komputer setelah Anda mengaktifkannya. Fungsi BIOS antara lain yaitu:

Menjalankan dan juga memuat sistem operasi.

Pengujian terhadap perangkat keras (proses yang disebut dengan Power On Self Test, POST) serta Inisialisasi (penyalaan).

Mengatur berbagai macam konfigurasi dasar bagi komputer Anda, seperti waktu, tanggal, kinerja, konfigurasi proses booting, konfigurasi media penyimpanan dan juga stabilitas komputer.
