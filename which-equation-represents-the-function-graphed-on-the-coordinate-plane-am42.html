  2x  = [tex]2x^{2}[/tex] – 6x

4. Find the coordinate of the point: ➡️ (i) whose ordinate is -4 and which lies on x-axis ➡️(ii)whose abscissa is 2 and which lies in the y-axis


First, we must agree that

- abscissa is x value (the first value), and ordinate is y value (the second value), in the coordinate system.

- the point lies on x-axis, it means the value of y or the ordinate value is equal to zero, and also the point lies on y-axis, it means the value of x or the value of abcissa is equal to zero


So,

(i) whose ordinate is -4 and which lies on x-axis,

it means the y value of coordinate system must be equal to -4, in other word, any point that lies on line y=-4, which cannot lie on x axis (y=0)


(ii)whose abscissa is 2 and which lies in the y-axis,

it means the x value of coordinate system must be equal to 2, in other word, any point that lies on line x=2, which cannot lie on y axis (x=0),


but if you want to switch the word of problem "lie on x or y axis" above (I) and (II), I think I'm sure you can solve the problem easily now, after my explanation above.

8. On a map, 1 cm represents 60 km. Find the length on the map represents a distance of 480 km!​


Answer:

8 cm

Explanation with steps:

scale = map length : actual length

scale = 1 cm : 6.000.000 cm

scale = 1 : 6.000.000

48.000.000 cm / 6.000.000 = 8 cm

hence, 480 km is represented by 8 cm on the map
