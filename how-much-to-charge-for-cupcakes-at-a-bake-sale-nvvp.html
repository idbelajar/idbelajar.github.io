When he baked 36 more cupcakesin the afternoon, the number of chocolate Cupcakes 3/8 of thetotal number of cupcakes(a) How many cupcakes did Joe bake in the morning?(b) How many more blueberry cupcakes did Joe bake in the afternoon?​


Jawab:

(a) 180 cupcakes

(b) 27 more

Penjelasan dengan langkah-langkah:

In the morning Joe baked b + c cupcakes. Relations between b and c are:

b = c + 36                 (i)

b = 3/2 c                  (ii)

Put it in c, c= 2/3b

replace c in (i)

b = 2/3b + 36

1/3 b = 36

b = 108

c = 72

So, in the morning Joe bake 180 cupcakes.

Joe baked 36 more, so become 216 and c = 3/8 × 216 = 81.

b = 5/8 × 216 = 135

Joe baked 135 - 108 blueberry cupcakes in the afternoon = 27.
