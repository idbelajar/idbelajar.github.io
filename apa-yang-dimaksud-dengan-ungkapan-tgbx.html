1. lim x ➡ o sin 5 x+7x-tg2x/tg9x-sin4x​


Jawab:

limit terigonometri bentuktak tentu

Penjelasan dengan langkah-langkah:

lim(x-> 0) { sin 5x  + 7x - tan 2x } / [ tan 9x - sin 4x }

*bagi dengan x*

= lim(x-> 0) { sin 5x/x  + 7x/x - tan 2x /x} / [ tan 9x/x - sin 4x/x }

= { 5 + 7 - 2 } /  { 9 - 4 }

= 10/5

= 2
