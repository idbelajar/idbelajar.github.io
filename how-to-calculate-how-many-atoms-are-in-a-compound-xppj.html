1. How many calcium atoms are in 16,2 of calcium


We're asked to calculate the number of atoms of Ca in 16,2 g Ca.

What we must first do is convert the given mass of calcium to moles of calcium, using its molar mass (referring to a periodic table, this is 40.08 [tex]\frac{g}{mol}[/tex] )

= 16,2 g Ca [tex](\frac{1 mol Ca}{40,08 g Ca} )[/tex]

= 0,404 mol Ca

Using Avogadro's number, 6,022 × 10²³ [tex]\frac{particles}{mol}[/tex] we can calculate the number of atoms present :

= 0,404 [tex](\frac{6,022 x 10^2^3atoms Ca}{1 mol Ca} )[/tex]

= 2,43 × 10²³ atom Ca

Sulfur S is a nonmetallic that is present in coal is bumed, sulfur is converted to sulfur dioxide and eventually to sulfuric acid that gives rise to the acid rain phebomenoun. how many atoms are in 16,3 gram of S​

Jawaban:

Jumlah mol S= [tex]\frac{massa}{mr}[/tex]

Jumlah mol S= [tex]\frac{16,3 gram}{32 g/mol}[/tex]

Jumlah mol S = 0,51 mol

Jumlah atom/Partikel S sebanyak = 0,51 mol x 6,02. 10²³ = 3,07.10²³

Jadi, jumlah atom s sebanyak 3,07.10²³ atom

Penjelasan:

Hubungan Jumlah Mol dengan Jumlah Partikel

Satu mol zat menyatakan jumlah partikel yang terkandung di dalamnya yang dinyatakan dalam satuan bilangan Avogadro = 6,02 x 1023  partikel  

RUMUS KONSEP MOL

Dengan demikian, 1 mol zat mengandung 6,02 x 1023 partikel. Hubungan antara jumlah mol dan jumlah partikel dapat dirumuskan sebagai berikut:

Jumlah mol X (n) = jumlah partikel X/ L

atau

Jumlah partikel X = n x L

Bilangan Avogadro (L) ditemukan oleh Johann Loschmidt pada 1865. Nama Avogadro dipilih sebagai penghormatan kepada Avogadro karena beliau orang pertama yang mengusulkan perlunya satuan jumlah partikel. Adapun nama Loschmidt diabadikan sebagai simbol bilangan tersebut, L.

Nilai bilangan Avogadro adalah L = 6,02 x 1023

9. how to calculate this?


[tex]P(X) = 0.27 \\
P(Y) = 0.48 \\
P(X|Y') = 0.36[/tex]
The last one called conditional probability.

If [tex]P(X) = 0.27[/tex] then
[tex]P(X') = 1-P(X)
\\P(X')=1-0.27
\\P(X')=0.73[/tex]

If [tex]P(Y) = 0.48[/tex] then
[tex]P(Y') = 1-P(Y)
\\P(Y') =1-0.48
\\P(Y') =0.52[/tex]

a. [tex] P(X \cap Y')[/tex]?
We can use [tex]P(X|Y') [/tex] to help us find [tex] P(X \cap Y')[/tex].

[tex]P(X|Y') = \frac{P(X \cap Y') } {P(Y')} \\ 0.36 = \frac{P(X \cap Y')}{0.52} \\
P(X \cap Y')= (0.36)(0.52)= 0.1872 \\ [/tex]

b. [tex]P(Y|X') [/tex]?
[tex]P(Y|X') = \frac{P(Y \cap X') } {P(X')} \\ [/tex]
First we need to calculate [tex]P(Y \cap X')[/tex], but we need to know that
[tex]P(X) = P(X \cap Y) +P(X \cap Y') \\
0.27 = P(X \cap Y) + 0.1872 \\
P(X \cap Y) = 0.27 - 0.1872 = 0.0828[/tex]

And,
[tex]P(Y) = P(Y \cap X) +P(Y \cap X') \\
0.48 = 0.0828 + P(Y \cap X') \\
P(Y \cap X') = 0.48 - 0.0828 = 0.3972[/tex]

Then,
[tex]P(Y|X') = \frac{P(Y \cap X') } {P(X')} \\
= \frac{0.3972} {0.73} \\
= 0.5441[/tex]

c. Two events X and Y are independent if and only if [tex]P(X \cap Y) = P(X) P(Y)[/tex]

Now lets check the value of both side,
[tex]P(X \cap Y) = 0.0828[/tex]
[tex]P(X)P(Y)=0.27×0.48 =0.1296[/tex]

Both values are different then we can conclude that both events X and Y are not independent because [tex]P(X \cap Y) ≠ P(X) P(Y)[/tex].
