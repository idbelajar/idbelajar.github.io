8. You . . . permit to your mother if you go to schoolA. MustB. WouldC. MayD. Mustn’t​


Jawaban:

A. MUST

Penjelasan:

karena :

Kamu .... permisi ke ibumu jika kamu ingin pergi sekolah

maka jawabannya A.MUST(harus)

kenapa tidak B.WOULD?

bahasa indonesia dari Would Adalah AKAN

kenapa tidak C.MAY?

karena bahasa indonesia dari MAY Adalah MUNGKIN

kenapa tidak D.MUSTN'T?

arti nya sama dengan MUST tetapi ada kata (N'T) di belakang kata, maka MUST= Harus

Dan MUSTN'T = TIDAK HARUS

(Karena "n't" singkatan dari Tidak)

║CONDITIONAL SENTENCE║

[tex]Answer[/tex]

You must permit to your mother if you go to school.

{ Kamu harus minta izin pada ibu mu jika kamu pergi ke sekolah }

The correct answer is A. must

[tex]Discussion[/tex]

Pernyataan diatas termasuk kalimat conditional { If clause + reminder ) yang merujuk pada mengingatkan suatu hal yang harus dilakukan demi kebaikan. Hal ini fakta apa adanya yang terkait di sekitar kita bahwa setiap keluar rumah, diharuskan minta izin terlebih dahulu kepada orang tua demi kebaikan yaitu  agar orang tua tidak khawatir.

[tex]Explanation[/tex]

Penggunaan -If clause terdiri dari 5 jenis

1. If Clause + reminder ( pengingat )

If clause ini digunakan untuk mengingatkan terhadap suatu kebaikan atau hasil tertentu demi meminimalisir kejadian yang tidak diinginkan.

Rumus

IF+simple present, Subject+has to/have to/must + verb 1 + O/C

Example

If you want to pass the last exam, you must study hard.

{ Jika kamu ingin lulus ujian akhir, kamu harus belajar giat }

2. If Clause+ Suggestion ( saran )

If clause ini digunakan untuk memberikan masukan (saran), langkah-langkah yang seharusnya dilakukan.

Rumus

If+simple present, Subject+should/had better/ought to + verb 1  + O/C

or

If+simple present, why don't + subject + verb 1 + O/C

Example

If you get toothache, you should visit a dentist.

{ Jika kamu sakit gigi, sebaiknya kamu menemui dokter gigi }

3. If Clause + Imperative  ( Perintah )

If clause ini digunakan untuk memerintah seseorang melakukan sesuatu atas hal tertentu yang diinginkan

Rumus

If + simple present, Verb 1 + O/C

Example

If you want to have slim body, eat vegetables and low fat foods.

{ Jika kamu ingin memiliki tubuh langsing, makanlah sayuran dan makanan rendah lemak jenuh }

4. If Clause + General Truth

If clause ini digunakan untuk menjabarkan fakta yang tidak bisa disangkal kebenarannya.

Rumus

If+simple present, simple present

Example

If you put water in the freezer, it changes into an ice.

{ Jika kamu memasukan air ke dalam lemari pendingin, itu berubah menjadi es }

5. If Clause of showing a dream ( Angan/mimpi )

If clause ini digunakan untuk menjabarkan sebuah keinginan / hasrat impian di masa mendatang.

Rumus

If + simple present , simple future

Example

If I finish my study, I will go abroad

{ Jika saya lulus sekolah, saya akan pergi ke luar negeri }
