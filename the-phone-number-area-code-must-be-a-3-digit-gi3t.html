3. a two digit positive number is such that the ones digit is 2.5 times as much the tens digit. if the diffrence beetween the number and the number obtain when the digit are reversed is 27


Solution:

Let the both digits are a and b such that
a*b is the required number with b is ones
digit and a is tens digit.
then we have
a*b - b*a = 27
and
b = 2,5 a .....(1)

Now, we solve the above system of the equations:

a*b = (a×10) + (b×1) = 10a + b
b*a = (b×10) + (a×1) = 10b + a

|a*b - b*a|= 27
|10a + b - (10b + a)| = 27
|9a - 9b|= 27
|a - b|= 3 ...... (2)

of (1) and (2), substitute equation (1)
to equation (2), we get
|a - b|= 3
|a - 2.5a| = 3
| -1.5 a| 3
1.5 a = 3
a = 2
and b = 2,5 a = 2.5×2 = 5
therefore a*b = 10a + b = 10×2 + 5 = 25
and b*a = 10b+a = 10×5 + 2 = 52


Answer:
the number is 52 and the reversely obtained number is 25.
