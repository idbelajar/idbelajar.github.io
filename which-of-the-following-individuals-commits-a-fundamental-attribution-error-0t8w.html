5. Confidence Interval Write the, 1. Critical Value 2. Margin of Error 3. Confidence Interval Of the following problem a) A cookie factory produced 32 flaky cookies in every 100 cookies, What is the proportion of the flaky cookies of the population if the confidence level is 95%?


Jawaban:

Penjelasan:

[tex]CI = \hat{p} \pm Z^{*} \sqrt{\dfrac{\hat{p}\cdot q}{n}}\to q = 1-\hat{p}\\\text{Z$^{*}$ = Critical Value} \\\\Z^{*} \sqrt{\dfrac{\hat{p}\cdot q}{n}} = \text{MoE} = \text{Margin of Error}\\\\CI = \hat{p} \pm MoE[/tex]

Jika CL = 95% maka Z* ≈ 1.96

[tex]\boxed{\bold{Z^{*}= 1.96}}[/tex]

[tex]CI = \hat{p} \pm 1.96 \sqrt{\dfrac{\hat{p}\cdot q}{n}}[/tex]

[tex]o = 32, n = 100[/tex]

[tex]\hat{p} = \dfrac{o}{n} = \dfrac{32}{100} = \dfrac{8}{25} = 0.32\; , q =1-\hat{p}= \dfrac{17}{25} = 0.68[/tex]

[tex]MoE = 1.96\cdot \sqrt{\dfrac{\hat{p}\cdot q}{n}}\\\\MoE = 1.96 \cdot \sqrt{\dfrac{\dfrac{8}{25} \cdot \dfrac{17}{25} }{100}}\\\\MoE = \dfrac{ 1.96}{10\cdot 25}\cdot \sqrt{8\cdot 17}\\\\MoE = \dfrac{ 1.96}{125}\cdot \sqrt{34}\\\\\boxed{\bold{MoE \approx 0.0914}}[/tex]

[tex]CI = \hat{p} \pm MoE\\\\\boxed{\bold{CI = 0.32 \pm 0.0914}}\\\\\boxed{\textbf{$\bold{CI = 0.4114}$ atau 0.2286 }}[/tex]
