15p/5 = 6

Jawab:

Range: 7,3%

Mean deviation: -0,74%

Standard deviation: 1,57%

Coefficient of variation: 22%

Penjelasan dengan langkah-langkah:

Range: Range adalah selisih antara nilai terbesar dan terkecil dari data. Dalam kasus ini,

range = 11,6% - 4,3%

          = 7,3%.

jadi, range : 7,3%

Mean deviation:

Nilai rata-rata dari data tersebut adalah

(4,3% + 4,9% + 7,2% + 6,7% + 11,6%) / 5

= 7,14%

Selisih antara setiap nilai dengan nilai rata-rata adalah:

4,3% - 7,14% = -2,84%

4,9% - 7,14% = -2,24%

7,2% - 7,14% = 0,06%

6,7% - 7,14% = -0,44%

11,6% - 7,14% = 4,46%

Mean deviation

(-2,84% + -2,24% + 0,06% + -0,44% + 4,46%) / 5

= -0,74%.

jadi, Mean deviation: -0,74%

Standard deviation

Rumus untuk menghitung standard deviation adalah:

SD = √(((x1 - x̄)^2 + (x2 - x̄)^2 + ... + (xn - x̄)^2) / (n - 1))

dimana:

x1, x2, ..., xn = nilai-nilai dari data

x̄ = nilai rata-rata dari data

n = jumlah data

Jadi, standard deviation dari data tersebut adalah:

SD = √(((-2,84% - (-0,74%))^2 + (-2,24% - (-0,74%))^2 + (0,06% - (-0,74%))^2 + (-0,44% - (-0,74%))^2 + (4,46% - (-0,74%))^2) / (5 - 1))

= √((3,1^2 + 1,5^2 + 0,8^1 + 0,3^2 + 5,2^2) / 4)

= √(9,61 + 2,25 + 0,64 + 0,09 + 27,04) / 4

= √(39,54) / 4

= 6,27% / 4

= 1,57%

jadi, Standard deviation: 1,57%

Coefficient of variation

standard deviation/Nilai rata-rata dari data

1,57% / 7,14%  

= 0,22 atau 22%.

jadi, Coefficient of variation: 22%
