9. If 2 is added to the numerator and 3 to the denominator of a fraction, the value obtained is 3/5. If 2 is subtracted from the numerator and 3 from the denominator of the original fraction, the value is 4/7. Find the original fraction.


Jawab:

Suppose the original faction can be written as a/b where a and b is co-prime. Then, write the problem in the following equation.

a+2/b+3 = 3/5 .....(1)

a-2/b-3 = 4/7 ..... (2)

Then, simplify each of the equation.

(1) a+2/b+3 = 3/5

       5(a+2) = 3(b+3)

       5a+10 = 3b+9

       5a-3b = -1 ..... (3)

(2) a-2/b-3 = 4/7

       7(a-2) = 4(b-3)

        7a-14 = 4b-12

       7a-4b = 2 .....(4)

We can use either substitute. eliminate, or combine both. In this case,          Sa-chan will be using Elimination. Eliminate (3) and (4)

5a-3b = -1                  |x4| = 20a-12b = -4

7a-4b = 2                  |x3| =  21a-12b = 6

--------------------------------------------------------- -

                                              -a=-10

                                               a=10

We get a=10. Now, substitute a to either one of the equation. Sa-chan will be using the 2 equation.

a-2/b-3=4/7

10-2/b-3=4/7
