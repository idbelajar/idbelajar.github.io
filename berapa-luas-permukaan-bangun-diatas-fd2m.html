10. 2. perhatikan bangun berikut.A. bangun apa saja yang menyusun bangun di atas?B. tentukanlah luas permukaan daru bangub di atas?​


Jawab:

a. bangun setengah bola dan sebuah tabung.

b. luas permukaan setengah bola = 1/2 x 4 x π x r²

   luas = 1/2 x 4 x 22/7 x 7²

   luas = 2 x 22/7 x 49

   luas = 2 x 154

   luas = 308 cm²

   luas permukaan tabung = 2 x π x r x (r + t)

   luas = 2 x 22/7 x 7 x (7 + 10)

   luas = 2 x 154 x 17

   luas = 308 x 17

   luas = 5.236 cm²

lalu, kita jumlahkan kedua hasil di atas, maka akan di peroleh hasil sebagai berikut =

luas permukaan bangun gabungan = 308 + 5.236

luas permukaan bangun gabungan = 5.544 cm²

jadi, luas permukaan bangun gabungan pada gambar di atas adalah 5.544 cm²
