8. Induksi MatematikaAn ATM (Automated Teller Machine) only provides Rp 20.000 and Rp 50.000. What kind of money multiplier which can be issued by the ATM?Prove your answer by induction.Note: Rp stands for Rupiah (the currency in Indonesia).


Let k = 20000x + 50000y for all x, y in non-negative integer.
20000x + 50000y = 10000(2x + 5y)= 10000(2(x + y) + y)
For any non-negative integer (x + y) it'll hold on even number and for any non-negative integer y it'll hold on odd number. Thus, 20000x + 50000y = 10000z for z in non-negative integer

Thus, we could assume that 20000x + 50000y = 10000n, n ≥ 4

We will prove that 10000 is the multiplier that can be issued by the ATM with the minimum of 40000

let P(n) :  10000n = 20000x + 50000y for some x, y and n ≥ 4

P(4) : 10000(4) = 40000 = 20000(2) + 50000(0)

Induction hypotesys : let P(k) true
thus,
P(k + 1) = 10000(k + 1) = 10000k + 10000 = 10000k + (50000 - 40000)= 20000x + 50000y + (50000(1) - 20000(2))= 20000(x - 2) + 50000(y)
Thus, proved.

14. 131. Semakin besar MPC, semakin ..... A. besar pula MPS B. kecil multiplier effect yang disebabkan oleh kenaikan pengeluaran pemerintah C. besar multiplier effect yang disebabkan oleh kenaikan pajak D. besar pula koefisien balance budget multiplier ​


Pendahuluan

Secara umum, untuk memperoleh rumus multiplier effect dapat digunakan cara yang terdapat digambar di atas. Multiplier pajak (Tx) dapat dihitung dengan rumus b/(1-b), dengan b = MPC. Sedangkan pengeluaran pemerintah (G) dapat dihitung dengan rumus 1/(1-b).

Pembahasan

A. Semakin besar MPS

MPS dapat dihitung dengan rumus MPS = 1 - MPC

Misal,

MPC = 0.6, maka MPS = 1 - 0.6 = 0.4

MPC = 0.8, maka MPS = 1 - 0.8 = 0.2

Jadi, dari dua permisalan diatas, ternyata diperoleh bahwa semakin besar MPC, ternyata MPS akan semakin kecil. Jadi jawaban A adalah salah.

B. Semakin kecil multiplier effect yang disebabkan oleh kenaikan pengeluaran pemerintah

Multiplier untuk G adalah 1 / (1 - b)

Kita misalkan,

MPC = 0.6, maka k = 1 / (1 - 0.6) = 2.5

MPC = 0.8, maka k = 1 / (1 - 0.8) = 5

Jadi, dari dua permisalan diatas, diperoleh hasil bahwa semakin besar MPC, maka koefisien multiplier untuk pengeluaran pemerintah akan semakin besar. Jadi jawaban B juga salah.

C. Semakin besar multiplier effect yang disebabkan kenaikan pajak

Multiplier untuk Tx adalah b / (1 - b)

Kita misalkan,

MPC = 0.6, maka k = 0.6 / (1 - 0.6) = 1,5

MPC = 0.8, maka k = 0.8 / (1 - 0.8) = 4

Jadi, dari dua permisalan diatas, diperoleh hasil bahwa semakin besar MPC, maka koefisien multiplier untuk kenaikan pajak akan semakin besar juga. Jadi jawaban C adalah benar.

Kesimpulan

Jawaban yang benar adalah C. Semakin besar multiplier effect yang disebabkan oleh kenaikan pajak


15. Pemerintah memutuskan untuk melakukan pemotongan pajak. apabila diketahui mpc sebesar 0.75. tentukan berapa multiplier effect-nya!​


Diketahui : MPC = 0,75.

→ MPS = 1 - MPC = 0,25.

→ Multiplier Effect untuk Pajak = k = ΔY / ΔTx = [tex]\frac{-MPC}{1-MPC}[/tex] = [tex]\frac{-MPC}{MPS}[/tex]

Penyelesaian :

Multiplier Pajak (k tx) =  [tex]\frac{-MPC}{MPS}[/tex]  =  [tex]\frac{-0,75}{0,25}[/tex] =  -3.
