1. Why is it called Boolean? How does a Boolean work? What is 0 in Boolean? Is 0 True or false boolean?


Penamaan Boolean karena berasal dari nama matematikawan Inggris bernama George Boole, karena dia yang pertama kali mendefinisikan istilah itu.

Boolean merupakan tipe data yang memiliki dua nilai, true (1) dan false (0). Operasi dasar logika Boolean ada AND dan OR.

1 AND 1 = 1

0 AND 0 = 0

1 OR 0 = 1

0 OR 1 = 1
