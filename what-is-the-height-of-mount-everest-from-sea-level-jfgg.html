15. The average average height of a good is 142 cm theheightis 138 cm. what is the height of the other girls?​


Jawab:

Penjelasan dengan langkah-langkah:

Diketahui :

Rata-rata tinggi perempuan = 142 cm

x₁ = tinggi perempuan pertama = 138 cm

Ditanya : x₂

Jawab :

Rata-rata

=> (x₁ + x₂ ) / 2 = 142

=> (138 + x₂ ) = 2 * 142 = 284

=> x₂ = 284 - 138 = 146 cm

Tinggi perempuan kedua = 146 cm
