a = 1.000.000
b = 150.000
S12 = ...?

Sn = n/2(2a + (n-1)b)
S12 = 12/2(2.1000000 + (12-1)150000 )
       = 6(2000000 + 1650000)
       = 6(3650000)
       = 21.900.000

Bulan 9 = Gaji kedelapan + Gaji Tambahan = Rp2.050.000,- + Rp150.000,- = Rp2.200.000,-
