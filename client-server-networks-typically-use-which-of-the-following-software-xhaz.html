1. bagaimana pendapatmu tentang contoh software ftp client dan software ftp server​


Jawaban:

Aplikasi FTP Client:

1. FileZilla (Windows)

FileZilla menjadi salah satu software FTP Client yang cukup populer di kalangan pengguna Windows. Walaupun software ini juga dapat diaplikasikan pada Linux maupun Mac, sistem kerjanya memang lebih terintegrasi bila digunakan pada Windows.  

Keunggulan:

Sistem penggunaannya yang sangat mudah, yakni hanya dengan memasukkan alamat web, nama pengguna, dan kata kunci FTP, FTPS dan SFTP.  

Dapat mengupload file dalam format RAR, ZIP, dan EXE dengan kapasitas maksimal 4 GB.  

Pengiriman data yang fleksibel ke berbagai cloud provider seperti Google Drive, DropBox, dan OneDrive.

Kekurangan:

Bila kapasitasnya lebih dari 4 GB, kemungkinan besar proses mengupload akan memakan waktu lebih lama.

2. WinSCP (Windows)

Aplikasi FTP Client terbaik lainnya selain FileZilla adalah WinSCP. FTP Client yang dirancang khusus untuk Windows ini juga menjadi opsi yang sangat kuat berkat tampilan interface yang sederhana sehingga pemakaiannya menjadi sangat mudah.  

Keunggulan:

WinSCP mampu mengoperasikan beberapa protokol seperti SFTP, SCP, FTP, dan WebDev.

Dibekali dengan dukungan SSH password, keyboard-interactive, kunci publik, Kerberos (GSS) otentikasi, serta terintegrasi dengan Pageant yang didukung penuh oleh otentikasi kunci publik dengan SSH.  

Kekurangan:

Hanya dapat digunakan oleh pengguna sistem operasi Windows saja.

3. FireFTP (Windows, Mac OS X dan Linux)

Berbeda dengan FTP Software pada umumnya yang memerlukan beberapa tahap instalasi, penginstalan FireFTP hanya sebatas pengunduhan add-on pada browser Mozilla Firefox Anda. Namun, pastikan bila Mozilla Firefox Anda telah terupdate sebelum menginstal versi FireFTP terbaru.  

Keunggulan:

Sama dengan FileZilla, FireFTP juga bisa diaplikasikan bagi Anda para pengguna Mac dan Linux.  

Dirancang dengan interface yang user-friendly alias mudah digunakan.  

FireFTP selalu memberikan keamanan penuh dengan metode enkripsi terbaru termasuk SSL, SFTP, dan TLS.  

Menawarkan fitur kompresi file dengan MODE Z sehingga prosesnya menjadi lebih cepat dan singkat.  

Kekurangan:

Beberapa pengguna mengeluhkan bila proses kompresi file mereka kerap kali error dan lambat bila file tersebut melebihi kapasitas 4 GB.

Aplikasi FTP Server:

1. FileZilla Server

Aplikasi FileZilla Server ini gratis dan merupakan FTP Server yang bersifat open source. Source code FileZilla Server berada di SourceForge.net.  Beberapa fitur yang ada di dalam aplikasi ini adalah:  

Kompresi dengan DEFLATE (MODE Z) Enkripsi dengan SSL / TLS (untuk FTPS) Izin per pengguna pada sistem file Alat konfigurasi GUI Speed limits

2. Microsoft IIS

Microsoft IIS atau biasa disebut sebagai IIS hadir pada bulan Januari 2010 lalu yang ditambahkan oleh PierreG dan pada bulan Mei 2019 lalu aplikasi ini mengalami pembaharuan.

Internet Information Service atau disingkat IIS yang diperuntukkan Windows server merupakan sesustu yang aman, fleksibel, dan mudah untuk mengelola web server untuk menghosting apapun di web. Mulai dari streaming media hingga hosting aplikasi web, Microsoft IIS siap untuk menangani tugasnya.

3. Syncplify.me Server!

Server Syncplify.me! merupakan aplikasi FTP, FTPS, dan SFTP gratis yang dapat digunakan di sistem operasi windows. Aplikasi ini dibuat oleh Syncplify, Inc. Anda tidak perlu khawatir dengan aplikasi ini karena aplikasi ini sudah terjamin aman, powerful, dan sangat mudah untuk dikonfigurasi.

Beberapa fitur utama yang terdapat di dalam aplikasi ini adalah dukungan untuk FTP biasa, FTPS eksplisit (TLS), FTPS implisit (SSL / TLS), dan protokol transfer file SFTP, basic secure shell (SSH-2) serta mendukung file dengan ukuran yang besar (> 4 GB). APlikasi ini benar-benar gratis, termasuk dengan peng-update an dan peningkatan seumur hidup, khusus untuk penggunaan pribadi.

---------

Explanation  

Suffix adalah imbuhan yang diletakkan di belakang suatu kata.

-ful = penuh (noun+ful → adjective)

colorful = penuh warna

doubtful = penuh keraguan

hopeful = penuh harapan

joyful = penuh sukacita

*adjective dari value adalah valuable

-ive = pertaining to (verb+ive → adjective)

conclusive

decisive

impressive

reductive

noun+ial → adjective

commercial = komersil

martial = berhubungan dengan perang

partial = sebagian

spacial = berhubungan dengan ruang

*adjective dari debate adalah debatable

verb+ure → noun

closure = penutup

departure = keberangkatan

fissure = retakan

lecture = kuliah

*noun dari polite adalah politeness

verb+ion → noun

collection = koleksi

correction = perbaikan

exhaustion = rasa lelah

prediction = prediksi

verb+ance → noun

allowance = ijin

deliverance = keselamatan

repentance = pertobatan

vigilance = kewaspadaan

*noun dari brief andalah brevity

adjective+ness → noun

cheerfulness = keriangan

drakness = kegelapan

gentleness = kelembutan

smallness = kekerdilan

*noun dari honest adalah honesty

-ward = menuju ke (noun+ward → adjective)

setelah di cek di kamus semua ada  

downward = ke arah bawah

forward = maju

heavenward = menuju ke surga

homeward = menuju ke rumah

houseward = menuju ke rumah

-less = tidak (punya); lawan -ful (noun+less → adjective)

careless = ceroboh; tidak hati-hati

fearless = tidak mempunyai rasa takut

powerless = tidak mempunyai kekuatan

thoughtless = tidak mempunyai akal; tidak berpikir dulu

*grand sudah adjective
