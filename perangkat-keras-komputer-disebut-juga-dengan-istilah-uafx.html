3. Perangkat keras komputer disebut dengan istilah​


Jawaban :

Perangkat keras komputer disebut dengan istilah​ hardware.

◖ PEMBAHASAN ◗

● Pengertian Perangkat Keras.

Perangkat keras komputer adalah bagian dari sistem komputer sebagai

perangkat yang dapat diraba, dilihat secara fisik, dan bertindak untuk

menjalankan instruksi dari perangkat lunak ( software ). Perangkat keras

komputer juga disebut dengan hardware. Hardware berperan secara

menyeluruh terhadap kinerja suatu sistem komputer.

● Fungsi Perangkat Keras.

Berdasarkan fungsinya, perangkat keras terbagi menjadi :

1. Sistem Perangkat Keras Masukan ( Input Device System )

2. Sistem Pemrosesan ( Central Processing System/ Central Processing Unit   ( CPU )

3. Sistem Perangkat Keras Keluaran ( Output Device System )

4. Sistem Perangkat Keras Tambahan ( Peripheral/Accessories Device System )

5. istilah istilah pending perangkat keras dan lunak​ dalam komputer


Jawaban:

1. Perangkat keras (Hardware) yaitu perangkat komputer yang dapat dilihat secara fisik dan bisa dilihat serta dipegang

2. Unit Input Yaitu Bagian pada hardware yang dicolokkan pada komputer (untuk membantu fitur). contohnya keyboard

3. Unit Penyimpanan yaitu bagian pada hardware yang berfungsi untuk menyimpan data yang ada. contohnya Hardisk

4. Unit proses yaitu bagian pada hardware yang berfungsi untuk menjalankan proses pada komputer. contohnya VGA

5. Unit output yaitu bagian pada hardware yang menjadi tujuan munculnya unit input. contohnya monitor

6. Perangkat Lunak (Software) Yaitu Perangkat komputer yang tidak dapat dilihat secara fisik dan tidak bisa dipegang, biasanya berupa aplikasi.

7. install yaitu proses pemasangan software ke komputer berupa aplikasi

8. Uninstall yaitu proses pencopotan Software dari komputer

9. OS Yaitu sistem operasi suatu komputer. contohnya windows

10. download yaitu mengambil data dari web kekomputer lokal

11. upload yaitu mengirim data dari komputer ke web

12. Crash/Bug yaitu kendala atau error yang terjadi pada suatu software

13. Malware yaitu Software/virus yang dapat merusak kinerja dari komputer

14. Antivirus yaitu software yang dapat mencegah adanya malware pada komputer

15. Program Pengguna komputer yang menjembatani penguna kamputer Dengan perangkat keras di sebut


Jawaban:

Program pengguna komputer yang menjembatani penguna komputer dengan perangkat keras disebut software (perangkat lunak). Pengguna komputer adalah orang yang memberi perintah, sedangkan perangkat keras adalah penerima instruksi. Perangkat lunak adalah jembatan yang menghubungkan antara pemberi perintah dengan penerima perintah.

Penjelasan:

Perangkat lunak komputer dibagi menjadi dua, yakni perangkat lunak sistem dan perangkat lunak aplikasi.

Perangkat lunak sistem terdiri dari sistem operasi, bahasa pemrograman, dan program utility.

Perangkat lunak aplikasi terdiri dari beberapa program aplikasi pengolahan, seperti pengolah kata, pengolah angka, pengolah video dan audio, dan lain-lain.
