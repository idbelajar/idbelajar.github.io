9. mohon bantuannya The picture is a graph of quadratic function......​


Persamaan umum fungsi kuadrat :

f(x) = Ax² + Bx + C

Diketahui grafik fungsi f(x) melewati titik :

» (3 , 0)

=> f(3) = 0

=> A.(3)² + B.(3) + C = 0

=> 9A + 3B + C = 0 ( i )

» (–1 , 0)

=> f(–1) = 0

=> A.(–1)² + B.(–1) + C = 0

=> A - B + C = 0 ( ii )

» (0 , 3)

=> f(0) = 3

=> A.(0)² + B.(0) + C = 3

=> C = 3

Subtitusikan nilai C ke ( i ) :

9A + 3B + C = 0

9A + 3B + 3 = 0

9A + 3B = –3 ( iii )

Subtitusikan nilai C ke ( ii ) :

A - B + C = 0

A - B + 3 = 0

A - B = –3 ( iv )

Eliminasi ( iii ) dan ( iv ) :

( iii ) : 9A + 3B = –3

(iv) × 3 : 3A - 3B = –9

________________ +

12A = –12 => A = –1

Subtitusikan nilai A ke ( iv ) :

A - B = –3

–1 - B = –3 => B = 2

Substitusikan nilai A, B, dan C ke persamaan fungsi kuadrat :

f(x) = Ax² + Bx + C

f(x) = (–1)x² + (2)x + (3)

Jadi, persamaan fungsi kuadrat grafik tersebut adalah : f(x) = –x² + 2x + 3

12. Arrange the quadratic equations whose roots are: a. 2 and 10 b. -9 and -4 c. 1/2 and 6 d. -3 and 7


a. (x - 2)(x - 10) = 0

x² - 12x + 20 = 0

b. (x + 9)(x + 4) = 0

x² + 13x + 36 = 0

c. (x - 1/2)(x - 6) = 0

x² - 6x + 3 = 0

d. (x + 3)(x - 7) = 0

x² - 4x - 21 = 0

So the quadratic equations whose roots are 2 and 10, -9 and -4, 1/2 and 6, -3 and 7 are x² - 12x + 20 = 0, x² + 13x + 36 = 0, x² - 6x + 3 = 0, x² - 4x - 21 = 0 respectively
