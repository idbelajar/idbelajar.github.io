4. Error Analysis Roni is ridw bicycle right now, because it can reduce the polution ​


Jawaban:

Roni is riding bicycle right now, because it can reduce the polution.

ride --> riding

Penjelasan:

Present continous digunakan untuk menyatakan hal yang sedang terjadi di masa sekarang. Biasanya ditandai dengan kata at the moment, now, right now, etc.

Formula:

(+) S + to be (is, am, are) + V-ing

(-) S + to be (is, am, are) + not + V-ing

(?) To be (is, am, are) + S + V-ing + ?

Pada soal di atas menggunakan ride (v1), maka kita ubah menjadi v-ing. Sehingga ride --> riding.

Semoga membantu
