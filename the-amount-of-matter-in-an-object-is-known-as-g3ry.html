8. the amount of energy, E joules (j) stored in an object with a mass of m kg is given by the formula E = mgh + ½mv², where h meter is the height of the object above the ground, v meter s-¹ is the velocity of the object and g is taken to be 10 m s-2with ways pls :)​


(i) E = mgh + 1/2mv²

so,

E-mgh = 1/2 mv²

(E-mgh) × 2 = mv²

mv² = 2(E-mgh)

v² = (2(E-mgh)) /m

v = √(2(E-mgh) /m)

(ii) Knows :

m = 0,5kg

h = 2m

E = 100j

g = 10m/s²

Question :

Find the v

Solution :

V = √(2(E-mgh) /m)

V = √(2(100j - 0,5kg×10m/s²×2m) /0,5kg)

V = √(2(100j - 10j) /0,5)

V = √(2(90)/0,5kg)

V = √(180/0,5kg)

V = √360m/s

V = (√36 ×√10)m/s

V = 6√10m/s

so the velocity of an object is 6√10m/s

Design a function named fallingDistance that accepts an object’s falling time (in seconds) as an argument. The function should return the distance, in meters, that the object has fallen during that time interval. Design a program that calls the function in a loop that passes the values 1 through 10 as arguments and displays the return value.


Jawaban:

#include <iostream>

using namespace std;

float fallingDistance(int t) //in second

{

   float g = 9.8;

   float d = (0.5*g)*t*t;

   return d;

}

int main()

{

   for (int i=1; i<=10; i++)

   {

       cout << "When t=" << i << ", distance covered is ";

       cout << fallingDistance(i);

       cout << "m" << endl;

   }

   

   return 0;

}
