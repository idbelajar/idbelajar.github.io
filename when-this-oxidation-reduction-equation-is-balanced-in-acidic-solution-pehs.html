3. One solution of the quadratic equation x^2 + px + 12=0 is 3, then p=... and another solution is...


x² + px + 12 = 0
salah astu penyelesaian x = 3
maka
x² + px + 12 = 0
⇒ (3)² + p(3) + 12 = 0
⇒ 9 + 3p + 12 = 0
⇒ 3p = -21
⇒ p = -7

Sehingga
x² - 7x + 12 = 0
⇒ (x - 3)(x - 4) = 0
maka
x - 3 = 0 ⇒ x = 3
x - 4 = 0 ⇒ x = 4

Jadi, p = -7, dan penyelesaian yang satunya = 4
