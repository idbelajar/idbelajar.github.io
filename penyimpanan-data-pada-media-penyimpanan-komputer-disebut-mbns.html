1. penyimpanan data pada media penyimpanan komputer disebut


Penyimpanan data pada media penyimpanan komputer disebut computer data storage. Kemampuan menyimpan data pada komputer sangatlah penting, kita sering dibuat jengkel apabila kita gagal pada saat kita menyimpan data di komputer kita. Beberapa media penyimpan data pada komputer juga memiliki kapasitas yang berbeda-beda, seperti ukuran hardisk yang bervariasi mulai dari 500GB, 1 TB hingga 2 TB.

Secara umum media penyimpanan data bisa dibagi menjadi beberapa jenis berdasarkan komponennya, jenis dari media penyimpanan data tersebut antara lain adalah

1. Penyimpanan data menggunakan optical disk, seperti CD R, DVD R dan masih banyak lainnya.

2. Penyimpanan data menggunakan magnetik disk, seperti pada disket, hardisk.

3. Penyimpanan online, penyimpanan ini kita meyimpan file atau data kita pada media penyimpanan online dan kita membutuhkan internet untuk melakukannya.

Demikian pembahasan mengenai penyimpanan data pada media penyimpanan komputer, dalam penggunaannyaa media penyimpanan tersebut memiliki kapasitas yang berbeda-beda. 4. media penyimpanan data komputer yang dapat menyimpan data dengan kapasitas sangat besar dan ditempatkan di dalam komputer disebut ​


Jawaban:

Penyimpanan Magnetik (Magnetic Disk)

Penjelasan:

Media Penyimpanan Data pada komputer sangat bervariasi dan selalu berkembang mengikuti perkembangan zaman. Media ini berfungsi sebagai menyimpan berbagai jenis data dalam bentuk digital.

Macam-macam media penyimpanan data komputer yang dapat kita temui saat ini antara lain, Penyimpanan Magnetik, Penyimpanan Optikal, dan Penyimpanan Awan.

Penyimpanan Magnetik merupakan penyimpanan sekunder yang paling umum dipakai hingga saat ini, bahkan terkadang dianggap sebagai penyimpanan wajib/utama karena memiliki kapasitas yang sangat besar. Beberapa contoh di antaranya adalah Harddisk, Memory Card. Harddisk umumnya terdapat pada setiap Personal Computer (PC) pada bagian dalamnya (Internal Harddisk).

Penyimpanan Optical memiliki ukuran yang lebih ringan apabla dibandingkan dengan penyimpanan magnetik karena hanya bermodalkan laser bertenaga rendah untuk membacanya.

Penyimpanan Awan merupakan jenis penyimpanan paling baru di antara penyimpanan lain. Penyimpanan ini bisa dibilang sangat ringan karena tidak memiliki bentuk fisik. Modal utama dalam penyimpanan ini adalah adanya akses dengan internet, karena menggunakan fungsi unggah dan unduh.
