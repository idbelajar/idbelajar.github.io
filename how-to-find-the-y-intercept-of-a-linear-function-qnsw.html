13. ##Quiz #174 ##• english version• big point• easy==========find the function that has an asymptote at x = 5 and an x-intercept of (6,0)?​


Jawaban:

[tex]f(x) = {}^{a} log(x - 5)[/tex]

with a>0 and a≠1

Penjelasan dengan langkah-langkah:

if A function has a vertical asymptote x = k, then if the value of x=k is substituted into the function, the result will be undefined.

Vertical asymptote : x = k

f(k) = undefined value

One function that has a vertical asymptote is the Logarithmic function. With the general form:

vertical asymptote: x=k ⇔ x=5

[tex]f(x) = {}^{a} log(x - k) \\ f(x) = {}^{a} log(x - 5)[/tex]

with a>0 and a≠1

If the value of x=5 is substituted into the Logarithmic function, the result will be undefined.

[tex]f(5) = {}^{a} log(5 - 5) = {}^{a} log(0) = undefined \: (proven)[/tex]

Known the function has an intercept-x (6,0).

if x=6 ⇔ f(x)=0

[tex]f(x) = {}^{a} log(x - 5) \\ f(6) = {}^{a} log(6 - 5) = {}^{a} log(1) = 0 \: (proven)[/tex]

So, the function is:

[tex]f(x) = {}^{a} log(x - 5)[/tex]

with a>0 and a≠1


14. what is can ?the function of canthe example of can in a sentenceshow to use can


•can adalah bisa.
can and could merupakan salah satu bentuk dari 'modal verbs'.
•Fungsi can untuk bertanya(asking),possibility(kemungkinan),request(permintaan),offering(penawaran),ability(kemampuan),dll.
•
2.1 Possibility (Kemungkinan)

3.2.1.1 Kita menggunakan ‘can’ untuk membuat pernyataan umum tentang apa yang mungkin terjadi.

Contoh:

a. It can be very cold in the mountain.

[Bisa jadi sangat dingin di gunung.]

Pada kalimat ini, berarti kemungkinan besar atau terkadang faktanya akan sangat dingin di gunung.

b. Children can be very expressive.

[Anak-anak bisa jadi sangat ekspresif.]

2.1.2 Kita menggunakan ‘could’ untuk membuat pernyataan umum tentang apa yang mungkin terjadi dalam bentuk past tense (lampau) dari ‘can’.

Contoh:

a. It could be very cold in the mountain.

[Bisa jadi sangat dingin di gunung.]

Pada kalimat ini berarti sebelumnya biasanya sangat dingin di gunung tapi penutur tidak yakin kalau saat ini akan dingin di gunung.

c. The students could have arrived hours ago.

[Murid-murid harusnya datang sejam yang lalu]

2.1.3 Kita menggunakan could untuk menunjukan sesuatu yang mungkin terjadi di masa yang akan datang, tetapi tidak yakin hal tersebut akan terjadi.

Contoh:

a. If you don’t hurry you could be late.

[mungkin kamu akan terlambat jika kamu tidak bergegas.]
