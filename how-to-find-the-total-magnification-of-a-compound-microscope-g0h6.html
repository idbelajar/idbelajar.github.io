6. A compound contains 4.04% hydrogen, 24.24% carbon, and 71.72% chlorine. Relative atomic masses: H = 1, C = 12, Cl = 35.5 Relative molecular mass of the compound = 99 Given this information, find the empirical formula and the molecular formula of the compound.


Menentukan Rumus Empiris berarti kita menentukan perbandingan paling kecil dari tiap jumlah molekulpertama kita tentukan rumus molekul dengan membagi jumlah kadar atom dengan massa relatifnya

H = 4.04 ÷ 1 = 4,04 dibulatkan ke bawah → 4

C = 24.24 ÷ 12 = 2.02 dibulatkan ke bawah → 2

Cl = 71.72 ÷ 35.5 = 2.02... dibulatkan ke bawah → 2

Didapatkan rumus molekularnya adalah C2H2Cl2Kemudian perkecil perbandingannya maka didapatkan CH2Cl

Ar H = 1

perbandingan mol

C : H : O

66,7/12 : 11,1/1 : 22,2/16

5,558 : 11,1 : 1,387 (dibagi 1,387)

4 : 8 : 1

the empirical formula => C4H8O

molecular formula => (C4H8O)n = 72

=> (4×12 + 8x1 + 1x16)n = 72

=> (48 + 8 + 16)n = 72

=> 72n = 72

n = 72/72

= 1

if n = 1 the empirical formula = the molecular formula
